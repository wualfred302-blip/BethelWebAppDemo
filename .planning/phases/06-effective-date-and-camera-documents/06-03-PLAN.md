# Plan 06-03: Camera-First Documents

## Goal
Add `capture="environment"` attribute to the file input in ScanStep.tsx to open rear camera on mobile devices.

## Files to modify
- `src/app/apply/steps/ScanStep.tsx`

## Steps

### 1. Add capture attribute
- Locate the file input at line 92-98:
  ```tsx
  <input
    ref={fileInputRef}
    type="file"
    accept="image/*,.pdf"
    onChange={handleFileSelect}
    className="hidden"
  />
  ```
- Add `capture="environment"` attribute:
  ```tsx
  <input
    ref={fileInputRef}
    type="file"
    accept="image/*,.pdf"
    capture="environment"
    onChange={handleFileSelect}
    className="hidden"
  />
  ```

## Verification
- On mobile iOS/Android, tapping the upload area opens rear camera directly
- Desktop behavior unchanged (file picker still works)
- No change in accept behavior (still accepts image/*,.pdf)
